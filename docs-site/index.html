動物病院のための
オープンソース電子カルテ

VetCare Proは、予約・受付・カルテ・検査・会計・レセプトまで一通りの機能を備えた 動物病院向け業務支援システムです。PHP + SQLite で動作し、どなたでも無料でお使いいただけます。

ダウンロード (GitHub) 機能を見る 導入ガイド

重要：ご利用前に必ずお読みください

本システムは動物病院の実務経験がない監修者の指導のもと、AI（Claude）を使って構築されたプロトタイプです。
実際の動物病院でそのまま使用することは想定していません。

  • 会計・計算機能は正確性が保証されていません。税率計算、保険点数計算、請求金額等に誤りがある可能性があります。
  • レセプト（診療報酬明細書）機能は形式的なものであり、実際の保険請求で使用できる品質ではありません。
  • 薬品マスタ・検査マスタはサンプルデータであり、実際の薬品名・用量・基準値と異なる場合があります。
  • 本システムの利用により生じたいかなる損害についても、開発者・監修者は一切の責任を負いません。
  • 実運用する場合は、獣医師・動物病院スタッフの方が必ずすべてのデータと計算ロジックを検証してからお使いください。

監修者の経歴に関する重要な注意

本プロジェクトの監修者は公務員獣医師（行政機関勤務）です。動物病院（臨床）での勤務経験は一切ありません。

  • 動物病院の日常業務フロー（受付の流れ、会計処理、レセプト提出方法など）を実体験として知りません。
  • 「動物病院の電子カルテはこんな機能が必要だろう」という推測に基づいて仕様を決定しています。
  • 保険請求の実務、動物用薬品の用法用量、検査基準値の正確な数値は、臨床現場の専門家による検証が必須です。
  • 公務員獣医師としての業務（と畜検査、食品衛生監視、動物愛護行政等）と動物病院の臨床業務は全く異なります。

AI（Claude）で構築されたシステムです

本システムのソースコードの大部分は、Anthropic社のAIアシスタント「Claude」によって生成されました。 人間の監修者が仕様を指示し、AIがコードを出力する形で開発されています。

  • AIが生成したコードには、予期しないバグやセキュリティ上の問題が含まれる可能性があります。
  • 医療関連の業務ロジック（投薬量計算、検査基準値判定等）はAIが推測で実装しており、医学的な正確性は保証されません。
  • コードの品質向上・バグ修正のプルリクエストを大歓迎しています。
  • Claudeの出力はあくまでツールの成果物です。最終的な品質保証は利用者ご自身の責任となります。

Web予約システム

飼い主がスマホから24時間予約可能。枠の空き状況をリアルタイム表示。担当医師の確認、新患登録にも対応。

電子カルテ (SOAP)

SOAP形式の診療記録。テンプレート機能でよく使う記録を素早く入力。検査結果・画像の添付にも対応。

会計・請求管理 *

診療明細書・領収書の発行。消費税自動計算。保険連携機能とレセプト出力。* 計算精度は未検証

検査結果管理

血液検査等の結果を記録・基準値判定。異常値の自動ハイライト。CSV一括インポートにも対応。

患者・飼い主管理

患畜と飼い主をひも付けて管理。体重推移グラフ、画像管理、ワクチン履歴。複数ペットの一元管理。

セキュリティ

IPホワイトリストによるアクセス制限。CSRF対策、セッション管理。外部予約ページの分離設計。

会計・計算機能の不正確性

消費税の端数処理（切り捨て/切り上げ/四捨五入）の方式が実際の慣行と異なる可能性があります。 保険適用時の自己負担額計算ロジックは推測で実装されており、実際の動物保険制度と合致しない場合があります。 修正方法はカスタマイズガイドで詳しく解説しています。

レセプト機能の形式的な実装

レセプト（診療報酬明細書）は画面・印刷上の形式を整えただけであり、実際の動物保険のレセプト様式に準拠していません。 保険点数コード、点数テーブル、記載項目は実際の保険制度に基づいていないため、そのままでは保険請求に使用できません。

薬品・検査マスタの不正確性

サンプルデータとして登録されている薬品名・用法用量・検査基準値は、AIが一般的な資料から推測して作成したものです。 実際の動物用医薬品や検査基準値と異なる場合があるため、使用前に必ず最新の獣医学的根拠に基づいて修正してください。

セキュリティに関する注意

本システムはセキュリティの基本対策（CSRF対策、パスワードハッシュ化、IPホワイトリスト等）を実装していますが、 専門のセキュリティ監査を受けていません。個人情報を扱う場合は、必ず追加のセキュリティ対策を施してください。 インターネット上に公開する場合はSSL/TLS（HTTPS）の導入が必須です。
